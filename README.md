# OCML-DI: Offline Clinical Memory Layer with Drug Interaction Intelligence

## Overview
OCML-DI is a healthcare safety platform designed for African infrastructure realities. It enables portable patient records, drug interaction intelligence, and USSD access for feature phones.

## Features
- Encrypted QR medical wallet
- Offline drug interaction checks
- Risk scoring engine
- USSD access for feature phones
- Human-in-the-loop governance
- Audit logging and accountability

## Project Structure
- `backend/` → APIs, risk engine, USSD router
- `frontend/` → clinician dashboard, assets
- `design/` → branding, mockups, wireframes
- `docs/` → API documentation, demo script, judge FAQ
- `security/` → threat model, penetration test plan, checklist
- `data/` → drug interaction datasets
